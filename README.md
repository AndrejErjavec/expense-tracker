# Expense Tracker

Expanse tracker in React Native with web OTP bank CSV upload support

This repository is a Typescript monorepo (using internal packages approach) with the following packages:

**apps**

- mobile - Mobile app written in React Native
- web -
- server - Express server enabling CSV uploading, parsing and storing transactions in supabase

**packages**

- api - common supabase client, CRUD functions, auth functions and types
